# Codex Instructions — The Leimbach Method

## Immediate Setup Task

1. Run `python3 bootstrap_repository.py` from the repository root.
2. Confirm that the Nutrition, Training, Recovery, Tracking, Recipes, Assets, and Progress Logs directories were generated.
3. Confirm that every link in `README.md` points to an existing Markdown file.
4. Do not invent medical, nutrition, supplement, or training claims while performing repository setup.
5. Commit all generated files with the message: `Build publication-ready project framework`.
6. Leave `bootstrap_repository.py` in the repository for reproducibility.

## Editorial Standards

- Follow `STYLE_GUIDE.md`.
- Preserve Markdown front matter.
- Use grams for food portions and add cups only when useful.
- Mark unverified material as draft.
- Add evidence-based references before changing a chapter status to `final`.
- Keep personal health details limited to what is necessary for the coaching manual.

## Project Objective

Create a publication-ready personal performance manual for a 33-year-old, 6'4", approximately 260-lb strength athlete working rotating 3-to-7 day or night shifts, with a long-term objective of reaching a lean, strong 275 lb.
